# Workout

A workout tracker and instruction app: log workouts, browse an exercise
library with form instructions, build multi-day training programs, and
track progress over time.

## Features

- **Log Workout** — record sets, reps, weight, and duration for each exercise in a session.
- **Exercise Library** — searchable/filterable exercises with step-by-step instructions and form tips; add your own custom exercises.
- **Programs** — build multi-day training programs and start a pre-filled workout log from any program day.
- **Progress** — charts of max weight and volume over time per exercise, plus a dashboard summary.

All data is stored locally in the browser (`localStorage`) — no account or backend required.
